# Дашборд ДДС — инструкция по созданию

Создайте новый лист **«ДАШБОРД»** в таблице «ДДС РАБОЧАЯ 2026» и оформите его по шагам ниже. Все формулы рассчитаны на то, что данные лежат в листах **«ДДС: Сводный»** и **«Сводная по направлениям»**.

---

## Структура листа «ДДС: Сводный» (для формул)

- **Строка 1:** столбцы B1:M1 — номера месяцев 1–12.
- **Строка 219:** «Деньги на конец месяца» — **итого** по всем кошелькам (B219:M219).
- **Строки 220–225:** балансы по кошелькам (Сбербанк, Касса, Фонд Развития, Фонд Абонементов, Фонд Налоги, Фонд Мастер) — B220:M225. При добавлении кошельков строки продолжаются (226, 227, …).
- **Строка 16:** «Поступления от оказания услуг» — выручка по месяцам (B16:M16).

---

## Шаг 1. Дата «сегодня» и общий баланс на сегодня

1. В **A1** напишите: **Сегодня:**
2. В **B1** введите формулу даты:
   ``` 
   =TODAY()
   ```
   При необходимости задайте формат ячейки: Формат → Число → Дата.

3. В **A2** напишите: **Текущий остаток денежных средств:**
4. В **B2** — формула итого на конец **текущего** месяца (строка 219 на «ДДС: Сводный»):

   ```
   =INDEX('ДДС: Сводный'!$B$219:$M$219; 1; MONTH(TODAY()))
   ```

---

## Шаг 2. Баланс по каждому кошельку

Список кошельков и балансы на конец текущего месяца удобно вывести таблицей и потом использовать для графика.

1. В **A4** напишите заголовок: **Кошелёк**
2. В **B4** напишите: **Баланс на конец месяца**
3. Начиная с **A5**, перечислите кошельки (как на листе «ДДС: Сводный» в столбце A, в блоке «Деньги на конец месяца»). Например: Сбербанк, Касса, Фонд Развития, Фонд Абонементов, Фонд Налоги, Фонд Мастер.
4. В **B5** вставьте формулу и растяните её вниз по всем кошелькам (B6, B7, … B10 для шести кошельков):

   ```
   =IF(A5=""; ""; INDEX('ДДС: Сводный'!$B$220:$M$225; ROW()-4; MONTH(TODAY())))
   ```

   Пояснение:
   - Кошельки на «ДДС: Сводный» — строки 220–225. Первая строка данных на дашборде (5) → 1-я строка блока (220), поэтому `ROW()-4`.
   - Если список кошельков на дашборде начинается с другой строки, замените `4` на (номер этой строки минус 1).
   - Если кошельков больше шести, расширьте диапазон до $B$220:$M$231 (до 12 кошельков).

---

## Шаг 3. Выручка по месяцам (для графика и шкалы)

Выручку по месяцам можно брать из листа **«ДДС: Сводный»** (строка «Поступления от оказания услуг») или из **«Сводная по направлениям»**.

### Вариант А: данные с листа «ДДС: Сводный»

1. Строка **«Поступления от оказания услуг»** на «ДДС: Сводный» — **строка 16** (B16:M16).
2. На дашборде:
   - **D4:O4** — подписи месяцев (1, 2, 3, … 12) или «Январь» … «Декабрь».
   - **D5:O5** — выручка по месяцам.
3. В **D5** введите формулу и растяните вправо до **O5**:

   ```
   =IFERROR(INDEX('ДДС: Сводный'!$B$16:$M$16; 1; COLUMN()-3); 0)
   ```

   Столбец D даёт месяц 1, E — 2, … O — 12.

### Вариант Б: данные с листа «Сводная по направлениям»

Если выручка удобнее брать из «Сводная по направлениям» (столбцы по месяцам и «Итого»):

1. Найдите строку «Поступления от оказания услуг» (и при необходимости нужное «Направление бизнеса»).
2. В **D5** (месяц 1) формула по смыслу такая (подставьте свои лист и диапазон):

   ```
   =IFERROR(INDEX('Сводная по направлениям'!$C$4:$N$50; СТРОКА_ВЫРУЧКИ; 1); 0)
   ```

   И растяните на 12 месяцев, меняя последний аргумент INDEX (1…12) или используя `COLUMN()` аналогично варианту А.

Итог: на дашборде у вас есть строка/столбцы с выручкой по месяцам 1–12.

---

## Шаг 4. Шкала распределения поступлений по месяцам

«Шкала распределения поступлений за месяцы» — это визуализация выручки (или всех поступлений) по месяцам.

1. Оставьте подписи месяцев в **D4:O4** (1–12 или Январь–Декабрь).
2. Выручка по месяцам — в **D5:O5** (из Шага 3).
3. При необходимости добавьте строку **суммарных поступлений** (все приходы, не только выручка): тогда на «ДДС: Сводный» найдите строку с итого по «Поступление» по месяцам и по аналогии с Шагом 3 выведите её во вторую строку данных (например, D6:O6).

Дальше — график по этим данным (Шаг 5).

---

## Шаг 5. Графики на дашборде

### График 1: баланс по кошелькам (столбчатая диаграмма)

1. Выделите на дашборде:
   - столбец A (названия кошельков), например **A5:A10**;
   - столбец B (балансы), например **B5:B10**.
2. Вставка → Диаграмма.
3. Тип: **Столбчатая** (или «Гистограмма»).
4. Заголовок, например: **Распределение денежных средств по счетам**.
5. При необходимости задайте диапазон подписей по горизонтали: названия кошельков из A5:A10.

### График 2: выручка по месяцам (линейный или столбчатый)

1. Выделите:
   - подписи месяцев **D4:O4**;
   - выручку **D5:O5** (и при необходимости D6:O6 для «все поступления»).
2. Вставка → Диаграмма.
3. Тип: **Линейный** или **Столбчатый** — как вам удобнее для «шкалы распределения поступлений».
4. Заголовок, например: **Выручка по месяцам** или **Распределение поступлений по месяцам**.

При необходимости настройте подписи оси X (месяцы) и оси Y (суммы).

---

## Сводка формул (номера строк с листа «ДДС: Сводный»)

| Что | Ячейка | Формула |
|-----|--------|--------|
| Дата сегодня | B1 | `=TODAY()` |
| Итого на сегодня | B2 | `=INDEX('ДДС: Сводный'!$B$219:$M$219; 1; MONTH(TODAY()))` |
| Баланс кошелька | B5 и вниз | `=IF(A5=""; ""; INDEX('ДДС: Сводный'!$B$220:$M$225; ROW()-4; MONTH(TODAY())))` |
| Выручка месяц 1…12 | D5:O5 | `=IFERROR(INDEX('ДДС: Сводный'!$B$16:$M$16; 1; COLUMN()-3); 0)` |

Используемые строки на «ДДС: Сводный»:
- **219** — итого «Деньги на конец месяца»;
- **220–225** — балансы по кошелькам (Сбербанк, Касса, Фонд Развития, Фонд Абонементов, Фонд Налоги, Фонд Мастер);
- **16** — «Поступления от оказания услуг» (выручка).

После этого дашборд будет показывать общий баланс на сегодня, баланс по каждому кошельку, выручку по 12 месяцам и два графика.
